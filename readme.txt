== Description ==

This plugin adds Social Sharing Tools meta to the WordPress Classic and Block Editor.

It is built to make it fast and easy to set the way post/pages display when the link is shared to social media like Facebook, X (Formerly Twitter), LinkedIn, Discord and more.

= Easy to Use =
* When installed the Social Sharing Tools meta is automatically added to the Classic and Block editor.
* All necessary og:meta tags that are not already present are added to the sites header.
* Sharing Tools Meta includes a link to Free online Pablo-Buffer tool for creating a share image.
* Sharing Tool has live share previews for Facebook, LinkedIn, Discord and X (formerly Twitter) 
* Includes Facebook and LinkedIn Debug Tools.
